# template-custom-block
WordPress、カスタムブロック作成キッド

### 使用方法
1. WordPressのpluginフォルダにダウンロードしたフォルダを配置
2. packagejsonの内容をインストール

### カスタムブロック追加方法
src/blocksに移動以下のコマンドを入力
,,,,
npx @wordpress/create-block フォルダ名 --no-plugin
